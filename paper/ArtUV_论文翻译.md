## 摘要

UV 展开是计算机图形学中的一项基础任务,支撑着渲染管线中的各种视觉编辑操作。然而现有 UV 展开方法普遍存在耗时长、碎片化、缺乏语义、UV 岛不规整等问题,限制了实际可用性。一张**艺术家风格**的 UV 图不仅要满足基本要求——无重叠映射、最小失真——还要达到更高层次的标准:边界干净、空间利用率高、语义连贯。

本文提出 **ArtUV**,一个全自动、端到端的艺术家风格 UV 展开方法。它把专业 UV 制作流程拆解为两个阶段来模拟:**表面接缝预测**与**艺术家风格 UV 参数化**。接缝预测阶段使用 **SeamGPT** 生成语义上有意义的切割缝;参数化阶段则把优化式方法得到的粗糙 UV 与网格一起输入一个 **Auto-Encoder**,将其精化为艺术家风格的 UV 图。该方法保证语义一致性并保持拓扑结构,使 UV 图可直接用于 2D 编辑。在多个基准上的评测表明,ArtUV 是一个通用方案:既可作为专业渲染工具的插件无缝接入,也可作为独立系统快速生成高质量 UV。

## 1 引言

UV 参数化(即 UV 展开)是计算机图形学的基础任务,在现代渲染管线中扮演关键角色,是纹理编辑、光照贴图生成等众多下游应用的基石。具体而言,UV 展开要为网格的每个 3D 顶点 $(x,y,z)$ 建立到 UV 空间 2D 坐标 $(u,v)$ 的映射,同时保持顶点间的拓扑连接。一张实用的高质量 UV 展开既要满足**基本标准**(低失真、无重叠),也要满足**高层标准**(边界干净、空间高效、语义连贯)。

UV 展开方法通常可分为三类:**自顶向下**、**自底向上**(如 Bounded-distortion、OptCuts)与**学习式**(如 Nuvo、FAM):

- **自顶向下**:先在 3D 物体上确定接缝、把表面切分为独立图块(chart),再在尽量降低失真、保持边界整洁的前提下逐块展开,最后把这些 UV 岛高效打包成完整 UV 图;
- **自底向上**:从整个物体表面的离散元素出发,通过优化能量函数逐步合并三角形直至收敛,得到最终 UV 图;
- **学习式**:用循环映射网络做无监督训练——把 3D 物体投影到 UV 空间再重建回 3D,构成完整往返变换,过程中以映射双射性等物理约束引导网络优化;训练完成后,3D→2D 的前向通路即可直接给出 UV 展开。

三类方法各有痛点:**传统自顶向下**方法需要大量时间与专业经验——依赖资深美术师判断最佳切割位置、并在展开后手动调整 UV 岛,这种人工干预使失真与整洁度难以达到最优平衡。**自底向上**方法的离散化与重聚类过程常产生碎片化的 UV 图,破坏单个 UV 岛的整洁度。**现有学习式**方法则需要对每个 3D 模型进行漫长的训练,且通常以点云为输入以兼容各种格式——这会破坏点之间的拓扑关系,产生大量重叠、混乱不堪的 UV 映射,严重限制实用性。更根本的是,自底向上与学习式方法都**缺乏语义感知**:对角色模型,美术师希望四肢与躯干按语义分开展开,这能极大方便后续纹理编辑;缺少这种语义理解,自动化方法就无法给出直观、对美术师友好的结果。

本文提出一种**学习式的自顶向下** UV 展开方法,把任务分解为表面接缝预测与 UV 参数化两部分。接缝预测利用 **SeamGPT** 对原始网格表面做语义切分;UV 参数化则用一个 **Auto-Encoder 模型模拟人工调整 UV 的过程**——编码原始网格信息,并从整洁度、重叠、失真三方面约束输出的 UV 图。经过在精心构建的数据上的大量训练,得到一个通用的 UV 展开方法,能把传统建模软件的初始 UV 图优化成符合艺术家风格的 UV 图。最终形成一个全自动、端到端的 UV 展开方法,能在数秒内生成带语义信息的高质量艺术家风格 UV 图。

大量实验表明该方法优于传统与学习式 UV 展开方法。**主要贡献**:

1. 提出 **ArtUV**——一个新颖的学习式框架,能从 3D 网格自动生成兼具语义感知与专业质量标准的艺术家风格 UV 图;
2. 构建 **ArtUV-200K**——包含 20 万个艺术家风格 UV 岛的高质量数据集,填补了专注高质量 UV 图的数据集空白;
3. 全面评测表明 ArtUV 在失真、利用率、处理速度等指标上显著优于当前最先进的 UV 展开方法。

## 2 相关工作

### 2.1 基于优化的 UV 展开方法

传统 UV 展开方法主要依赖数值优化:在把 3D 网格表面映射到 2D 参数域时最小化特定能量函数,构成了早期乃至许多当代展开工具的核心。按是否预先给定表面接缝可分为两类:

- **给定接缝**(网格已切分为独立图块):LSCM 与 ABF++ 在 2D 域优化角度以逼近 3D 对应角,理论上实现角度失真最小的保角映射,但往往难以保证全局双射;SLIM、SCAF、Bijective Parameterization 等则把 UV 图初始化为双射映射,并在优化中显式加入局部/全局重叠约束以维持双射性。这类方法失真通常很低,但在同时优化边界规整度与失真最小化上存在固有困难。
- **不给定接缝**(联合优化切割与参数化):Bounded-distortion 方法把表面离散为围绕种子三角形迭代生长的紧凑三角面片,直到满足终止条件;OptCuts 提出联合能量函数,同步优化接缝长度与失真。但这类联合优化算法常因过度碎片化或缺乏语义连贯性而产出不实用的 UV 图。

因此,当前生产管线仍需大量人工干预——手工放置接缝、手动调整 UV 岛——才能满足专业渲染要求。

### 2.2 基于学习的 UV 展开方法

随着深度学习的发展,学习式 UV 展开成为研究热点。**Nuvo** 用多类别神经网络架构分别处理网格分割与参数化,借助多个损失函数强制双射性并最小化失真。**FAM**(Flatten Anything Model)提出物理启发的框架,包含表面切割、UV 变形、展开、打包等子网络,通过双向循环映射机制对任意 3D 表示实现点到点映射。但这些方法有明显局限:**缺乏语义**、需要**耗时的逐场景优化**;尤其 FAM 基于点云的做法会破坏拓扑结构、产生无法修复的重叠瑕疵。最关键的是,这些算法难以达到艺术家风格展开的整洁度与美感,UV 利用率欠佳,因而无法用于专业渲染管线。

### 2.3 切割缝预测方法

UV 展开通常包含表面切割与参数映射两步。受深度学习进展启发,一些工作(PIE-Net、EDC-Net、PCEDNet 等)用神经网络做边缘点检测,视其为逐点分类任务;EC-Net 将其重构为回归问题,学习残差点坐标与点到边距离以更精确地识别边缘点。在此基础上,**SeamGPT** 创新性地模拟专业工作流:用自回归网络把表面切割建模为"预测下一个切割点"的任务,能为美术师制作和 AI 生成的网格给出语义上有意义的切割。

## 3 数据准备(ArtUV-200K)

作者从多个开源 3D 数据集(Objaverse、Objaverse-XL、3D-FUTURE)收集带 UV 映射的网格模型,聚焦艺术家风格 UV 图,并实施了严格的筛选流程:

1. 从约 **35 万**个带纹理网格模型出发,先把完整模型拆解为独立的 **UV 岛**;
2. 过滤掉 UV 重叠或过度碎片化(单岛顶点数少于 5)的样本,得到约 **30 万**个独立 UV 岛;
3. 顶点数分布分析显示 **97% 以上**的 UV 岛顶点数少于 500;为提升训练稳定性、降低显存开销,剔除更大的岛(图 1:ArtUV-200K 顶点数分布);
4. 人工挑选语义信息明确、布局规整的 UV 岛;
5. 每个候选岛用 Blender 的 **Minimize Stretch(Ministretch)**方法处理,再用 **SSIM** 与原始 UV 图对比评分,**SSIM 在 0.5~0.8 之间**的岛被认定为"经过人工精调的高质量样本"入选。

## 4 方法

![图 2:ArtUV 流水线](2509.20710_source/figs_png/pipeline.png)

**图 2(ArtUV 流水线)**:先用 SeamGPT 预测输入网格表面上语义有意义的接缝;依据这些切割线把网格切分为图块;对每个图块,用 Ministretch-Unwrap(SLIM / SCAF / LSCM 一类优化式方法)快速生成初始 UV 图;再把初始 UV 图输入 ArtUV 参数化模块,优化为艺术家风格的 UV 图。

ArtUV 是一个同时涵盖**表面切割**与 **UV 参数化**的全自动艺术家风格 UV 展开方法:表面切割部分复现了先进的自回归接缝预测模型 SeamGPT;UV 参数化部分提出了一个基于 Auto-Encoder 的艺术家风格 UV 参数化模型。

### 4.1 预备知识:SeamGPT

给定 3D 网格 $M$(顶点集 $V\in\mathbb{R}^{N\times 3}$、三角面集 $F\in\mathbb{R}^{M\times 3}$),表面切割任务是找出切割顶点 $V_c\subseteq V$,它们沿网格边构成连通的接缝,把表面切分为离散图块。SeamGPT 把表面切割表述为**序列预测问题**:对切割顶点做空间排序与量化,每个 token 代表一个坐标值,**连续 6 个 token 定义一段接缝线段**。

具体流程:对输入 3D 网格,先在顶点与边上采样点云,用 Hunyuan3D 的点云编码器压缩为潜在形状条件;然后仿照 Meshtron 构建**沙漏式(hourglass)自回归解码器**——每层由多个 Transformer 堆叠组成,层与层之间通过保持因果性的下采样/上采样连接。解码器从 SOS 开始自回归生成坐标 token 直到 EOS,最后把离散 token 通过**最近点投影**吸附到网格表面得到接缝顶点。

### 4.2 ArtUV 参数化

![图 3:ArtUV 参数化模型架构](2509.20710_source/figs_png/architecture.png)

**图 3(参数化模型架构)**:Res-M MLP 对输入参数做基于重要性的维度映射;SAGEConv 通过图卷积融合相邻顶点的局部特征;Pyramid ED(金字塔编解码器)通过注意力编码器与由粗到细的解码器实现全局顶点交互;输出端把预测偏移与初始 UV 相加,得到最终 UV 参数化。

**建模(Modeling)**。UV 参数化的目标是建立从 3D 网格 $M$ 到 2D 平面 $P\in[0,1]$ 的连续**双射**映射,平面上对应的点集记为 $Q\in\mathbb{R}^{N\times 2}$。最直接的做法是把网格 $M$ 的全部信息 $I_M$——顶点 $V$、面 $F$、法线 $N$、度 $C$、曲率 $D$——直接作为输入,训练一个强模型端到端学习 3D→2D 映射。但大量实验表明,即便有 ground truth,直接学习该映射也非常困难;况且单纯的投影并非目标,最终要的是**整洁、低失真的艺术家风格 UV 图**。

因此,受传统自顶向下建模流程启发,作者用**优化式展开方法得到的 UV 图 $Q_i$ 作为初始化**,让参数化模型学习"如何把 $Q_i$ 调整成艺术家风格的 UV 图"——即把任务转化为**学习传统软件生成的 UV 与艺术家优化后的 UV 之间的差异**。$Q_i$ 也被并入输入,最终输入为 $I = I_M + Q_i$。模型预测初始 UV 图中每个顶点在"人工调整过程"中所需的**偏移量 $Q_o$**,最终 UV 图为:

$$Q_{pred} = Q_i + Q_o$$

**架构(Architecture)**。参数化模型由三部分组成:

1. **Res-M MLP**(带自适应维度映射的残差 MLP 模块):按各特征在 UV 展开任务中经验观察到的重要性($Q_i > V > C = N = D$)动态分配特征维度;残差结构在增强特征表示的同时有效保留输入的关键信息;
2. **SAGEConv 图卷积**:以顶点为节点、面邻接为边构建图结构,通过 SAGEConv 在相邻节点间传播局部特征,维持拓扑一致性;
3. **Pyramid ED 模块**:编码器中堆叠的注意力层实现全局顶点特征交互;由粗到细的解码器同时提取粗粒度全局结构与细粒度局部细节,预测 UV 空间偏移 $Q_o$;偏移与初始 UV 坐标 $Q_i$ 相加得到最终预测 $Q_{pred}$。

### 4.3 损失函数

为生成符合艺术家设计标准的 UV 图——**整洁、失真最小、无重叠**——用多项加权损失联合优化参数化模型。

**(1) UV 重建损失**。直接度量预测 UV 坐标 $Q_{pred}$ 与真值坐标 $Q_{gt}$ 的位置差异。计算前有一个关键预处理:先在旋转空间中把初始 UV 坐标 $Q_i$ 与真值 $Q_{gt}$ **对齐**(图 4)。对齐采用 **Horn 方法**(闭式解),通过最小化旋转差异求最优旋转矩阵 $R$:先计算 $Q_i$ 与 $Q_{gt}$ 的协方差矩阵

$$W=\sum_{i=1}^{N}(q_i-\bar{q})\cdot(p_i-\bar{p}) \tag{1}$$

其中 $q$、$p$ 分别是 $Q_i$ 与 $Q_{gt}$ 中的对应点。再对 $W$ 做奇异值分解求最优旋转:

$$W=U\Sigma V^{T} \tag{2}$$

$$S=\begin{bmatrix}1 & 0\\ 0 & \mathrm{sign}(\det(U)\cdot\det(V^{T}))\end{bmatrix} \tag{3}$$

$$R=USV^{T} \tag{4}$$

用 $R$ 把 $Q_i$ 对齐到 $Q_{gt}$ 后,重建损失为逐点 $L_1$ 距离:

$$L_{recon}=\left\|Q_{gt}-Q_{pred}\right\|_{1} \tag{5}$$

![图 4:UV 图对齐](2509.20710_source/figs_png/align.png)

*图 4:计算重建损失前,先用 Horn 方法把初始 UV 与真值 UV 在旋转空间对齐。*

**(2) 轮廓损失(silhouette loss,保边)**。为让预测 UV 图的结构规整度进一步接近人工展开结果,引入基于**可微渲染**的轮廓损失:渲染预测与真值 UV 图的轮廓图(图 5),计算二者的 $L_2$ 距离,引导模型关注最能体现 UV 岛整洁度的边界信息:

$$L_{silhouette}=\left\|Render_{gt}-Render_{pred}\right\|_{2} \tag{6}$$

![图 5:UV 图轮廓](2509.20710_source/figs_png/silo.png)

*图 5:通过可微渲染得到的 UV 图轮廓,用于约束 UV 岛边界的整洁度。*

**(3) 失真损失**。对每个三角面 $f$,计算其 3D→2D 映射的雅可比矩阵,做奇异值分解得到刻画拉伸强度的奇异值 $\sigma^1$、$\sigma^2$。失真损失定义为所有面上奇异值之差绝对值的(面积加权)平均,理想保角映射时趋于零:

$$L_{distortion}=\frac{1}{\sum_{f\in\mathcal{F}}|A_f|}\sum_{f\in\mathcal{F}}|A_f|\left\|\sigma_f^{1}-\sigma_f^{2}\right\|_1 \tag{7}$$

其中 $\mathcal{F}$ 是输入表面全部三角形的集合,$|A_f|$ 是面 $f$ 的面积。

**(4) 重叠惩罚**。为防止 UV 重叠影响后续纹理映射,基于法线方向设计惩罚项。关键观察:UV 域中发生重叠的三角面其**法线方向会翻转**。于是以法线为负方向的面数为惩罚:

$$L_{overlap}=\sum_{f\in F}\left(n_f\cdot\vec{z}<0\right) \tag{8}$$

其中 $F$ 为 UV 图中的面,$n_f$ 是面 $f$ 在 UV 图中的法向量,$\vec{z}$ 为参考观察方向。

**总损失**为各项的加权线性组合:

$$L_{total}=\omega_{r}L_{recon}+\omega_{s}L_{silhouette}+\omega_{d}L_{distortion}+\omega_{o}L_{overlap} \tag{9}$$

其中 $\omega_i\ (i=r,s,d,o)$ 为各损失项权重。

## 5 实验

在 ArtUV-200K 与 FAM 基准上做定性与定量评测:先把参数化模型与主流专业建模软件比较;再把完整 ArtUV 方法(覆盖从表面分割到 UV 参数化的全流程)与先进算法比较;最后通过消融实验验证损失函数设计的合理性与有效性。

### 5.1 实现细节

**SeamGPT**:作者咨询了 SeamGPT 原作者获取数据集与模型实现细节,并成功完整复现(详见附录)。

**ArtUV 参数化**:
- 初始化:Blender 的 Ministretch 算法;
- Res-M MLP 把输入 UV、顶点、法线、曲率、度的特征维度分别映射为 **128、62、32、32、32**;
- 经 **5 层 SAGEConv** 得到 512 维图特征;
- 注意力编码器:512 维、8 头、8 层;由粗到细解码器以 1/2、1/4 的比例下采样特征;
- 输出映射层用 **Tanh** 激活预测 2 维 UV 坐标,保证预测落在 $[-1,1]$;
- 训练:**24 块 H20 GPU(96GB)**,70 万步,batch size 32;
- 损失权重:$\omega_r=1.0$、$\omega_s=1.0$、$\omega_d=0.0001$、$\omega_o=0.01$;
- 推理:多数消费级 GPU 即可运行;模型面数少于 1000 时显存占用不超过 10GB。

在 ArtUV-200K 上(含 100 个多样化 3D 模型,每个都带**美术师手工标注的接缝**——从而排除接缝质量的干扰,直接比较展开效果)与专业建模软件对比。定量指标:**网格失真**(全部三角面的平均保角能量)与 **UV 利用率**(所有算法统一用 UVPackMaster 插件做布局优化)。

## 6 结论与局限

**结论**:本文提出 ArtUV,一个端到端生成艺术家风格 UV 图的方法。问题被分解为基于 SeamGPT 的表面分割,与"从初始 UV 图预测到艺术家风格结果的偏移量"的 UV 参数化。该方法能在数秒内生成整洁、有序、低失真的 UV 图,解决了当前专业工作流中耗时长、缺乏语义的问题。大量实验表明 ArtUV 在多项指标上优于现有方法,对下游应用的效率提升具有显著潜力。

**局限**(两点):

1. **对切缝质量高度敏感**:接缝不完整或不准确会在 UV 初始化时造成严重扭曲,即便输出仍保持干净的边缘,内部也会有明显形变;
2. **尚不支持 UV 岛复用(island reuse)**:复用岛若对齐不精确会造成严重重叠伪影,且岛复用会给模型训练引入额外复杂度。

**未来方向**:(1) 提升接缝质量与稳定性(如对高失真区域做二次分割);(2) 把 UV 岛复用整合进流水线(如基于相似度合并优化后的岛)。

## 快速要点总结(TL;DR)

1. **问题**:现有 UV 展开要么碎片化严重、缺语义(xatlas 等自底向上方法),要么逐模型优化极慢且拓扑混乱(Nuvo/FAM 等学习式方法);人工展开质量高但费时费力。"艺术家风格"= 无重叠 + 低失真 + 边界干净 + 空间利用率高 + 语义连贯。
2. **核心思想**:学习式自顶向下两阶段——**SeamGPT**(自回归 next-cut-point 预测)给出语义接缝并切块;每块先用优化式方法(Blender Ministretch)得到粗糙初始 UV $Q_i$,再由 **Auto-Encoder 参数化模型**(Res-M MLP → 5×SAGEConv → 金字塔注意力编解码器)预测顶点偏移 $Q_o$,输出 $Q_{pred}=Q_i+Q_o$,即"学习软件初始 UV 与艺术家精调 UV 之间的差异"。
3. **监督信号**:$L_1$ 重建(Horn 方法 SVD 旋转对齐后)+ 可微渲染轮廓损失(管边界整洁)+ 雅可比奇异值失真损失(管保角)+ 法线翻转重叠惩罚(管双射),权重 1.0 / 1.0 / 1e-4 / 0.01。
4. **数据**:自建 ArtUV-200K(35 万模型筛到 1.5 万物体 / 20 万 UV 岛;SSIM 0.5–0.8 判定"人工精调"样本)。
5. **效果**:ArtUV-200K 上失真 9.52 / 利用率 72.57%,双双超过美术师手工(10.90 / 70.08%),30 位美术师盲评艺术风格 4.22 分同样超过手工(4.12);FAM 基准上全流程 36 秒完成,失真 8.91、碎片 14 块,大幅优于 XAtlas(1292 块)与 Nuvo/FAM(约 49/94 分钟)。可作为 Blender/Maya/3DsMax 的插件使用。
6. **与 PartUV 的关系**:两者都在解决 UV 碎片化与语义问题,但路线不同——PartUV 是"语义部件树 + 几何启发式 + 递归搜索"的**非学习参数化**(ABF++),ArtUV 则是"SeamGPT 切缝 + 神经网络精调 UV 布局"的**学习式参数化**,直接以美术师成品为监督目标。
